# 💤 Sleep Quality Prediction Application

This project is a machine learning-based application designed to predict sleep quality using various lifestyle and health-related factors.

## 📌 Project Objective
The aim of this project is to analyze factors such as age, stress level, physical activity, heart rate, and sleep duration to estimate an individual's sleep quality. It demonstrates data preprocessing, model training, evaluation, and user interaction through an interface.

## 🚀 Features
- Multiple machine learning models for prediction  
- Interactive user interface (Gradio / HuggingFace Spaces)  
- Data analysis and visualization  
- Model comparison and performance evaluation  

## 🧠 Models Used
- Random Forest  
- K-Nearest Neighbors (KNN)  
- Support Vector Regression (SVR)  
- Decision Tree  
- Linear Regression  
- Artificial Neural Networks (ANN)  
- Convolutional Neural Networks (CNN)  
- Long Short-Term Memory (LSTM)  

## 📊 Dataset
The dataset includes the following features:

- Age  
- Gender  
- Occupation  
- Stress Level  
- Physical Activity  
- Daily Steps  
- Sleep Duration  
- Heart Rate  
- BMI (Body Mass Index)  
- Blood Pressure  

## 🛠 Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- TensorFlow / Keras  
- Matplotlib  
- Gradio
